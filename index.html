Carmus

Script for encoding, decoding, updating ID3 tags, converting cyrillic to "translit". And the same for file name.

This is my first try in Python.. And it's for managing MP3 files for my car. Script automaticaly trying to find correct encoding for ID3 tags. Takes data for tags from file name (if tags is empty or not even exist). Now it updates only "TIT2" and "TPE1", as I do not need other... Also converts cyrillic text in file name and ID3 tags to translit.


So,
Usage: carmus.py [options] file_name


Options:
--version show program's version number and exit
-h, --help show this help message and exit
-b, --backup do not remove original file
-u, --update update ID3 tags
-c, --convert convert all cyrillic text to translit
-r, --rename rename file in translit
-y, --yes automaticaly put "yes" in all dialogs
